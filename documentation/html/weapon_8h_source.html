3D FPS Game  1.00
computer game, created with OpenGL. The game is a FPS (first person sh0oter) - character moves in a three-dimensional maze, whose mission is to eliminate opponents and reach the end point. The camera is positioned with a first-person perspective (see only weapon forms). Map will find all sorts of bonuses that add as additional statistics (hit points, additional ammunition) .Map is loaded from a file, and the opponents will be deployed randomly around the map
